<h1 align="center">Currency web app </h1>


<div align="center">A prototype of a Web application for currency exchange. Users create an exchange request by selecting the required transfer currency and the currency of receipt and other data.</div>

---

## Motivation
Here I want to explore a few different tools for building web applications. Understand how they work and what underlies them. What are the differences between them. What are they made of. I don't want to know everything, my choice is due to the popularity of the framework, the work stack and personal preference. This application does not pursue commercial purposes, but is used as a learning base for the study and implementation of creative potential :)
